# Jogo do Número Secreto 🎮

Projeto desenvolvido durante um curso de JavaScript com o objetivo de praticar os conceitos fundamentais da linguagem.

## O que aprendi

Durante o desenvolvimento deste projeto, pratiquei:

- Variáveis (`let` e `const`)
- Condicionais (`if` e `else`)
- Funções
- Manipulação do DOM
- Eventos de clique
- Arrays (listas)
- Laços de repetição
- Operadores lógicos e de comparação
- Template Strings
- Boas práticas de organização de código em JavaScript

## Sobre o projeto

O jogo consiste em adivinhar um número secreto gerado aleatoriamente pelo sistema. A aplicação fornece dicas ao jogador até que o número correto seja encontrado.
